Cybersecurity

of the current  level of security of computer systems and networks, identification of existing
security problems,design anddevelopment of information security systems for organizations and
enterprises. The program will allow you to develop skills in software development, as well as
in technologies for protecting information and Informatization objects, ensuring information
integrity and analyzing system security
